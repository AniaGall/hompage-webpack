Składnia CSS to reguły

CSS odpowiada za prezentację, wygląd elementów HTML-a na ekranie, wydrukach...

Budowa reguły CSS:
selektor {
właściwość: wartość;
}

Znaczniki w CSS

< style > stylowanie, umieszczany w sekcji head,
< LINK > stworzenie zewnętrznego arkusza stylów, umieszczamy go w sekcji head:
< link href="style.css" rel="stylesheet" type="text/css" >
< header > nagłówek dokumentu/sekcji/artykułu,
< footer > stopka dokumentu, sekcji,
< h1-h6 > tekstowe nagłówki tytułowe,
< p > paragraf/akapit,
< br/ > złamanie linii,
< b > pogrubienie,
< i > kursywa,
< strong > pogrubienie, ma znaczenie semantyczne,
< em > kursywa, ma znaczenie semantyczne,
< a > tworzy łącza wewnątrz i zewnątrz dokumentu:
< a href="target" > opis łącza < /a >
< nav > nawigacja,
< img > obrazek o podanym adresie:
< img src= "" alt = "" />

Własności/właściwości w CSS

- background - tło,
- color - kolor,
